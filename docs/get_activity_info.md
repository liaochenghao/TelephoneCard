### 获取活动信息接口

**请求地址**:
```
    GET     api/v1/activity/{id}
```

**请求参数**:
```
    {
        "id": str  必填   默认参数为 1 
    }
```

**成功返回**：
```
{
    "code": 0,
    "msg": "请求成功",
    "data": {
        "id": 1,
        "text1": "12",
        "text2": "123",
        "text3": "123",
        "text4": "456",
        "text5": "46",
        "create_at": "2018-04-24T10:46:09Z"
    },
    "field_name": ""
}
```
